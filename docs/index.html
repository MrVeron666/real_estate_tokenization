Real Estate Tokenization Platform

Платформа для токенизации недвижимости на блокчейне Solana.

Бэкенд на Flask, смарт-контракты на Anchor.

Пример API вызова:

    curl -X POST http://127.0.0.1:5000/mint \
    -H "Content-Type: application/json" \
    -d '{"name":"Apartment #1","symbol":"APT1","supply":1000}'
